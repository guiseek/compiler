# Contributing

Clone this repository, access the repository in your editor and terminal of choice, then run the following commands.


#### Install

```sh
npm install
```

#### Build

```sh
npm run build
```

#### Link

```sh
npm link dist/
```


#### Quit

```sh
Ctrl + C
```
